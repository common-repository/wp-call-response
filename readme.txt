== Description ==
WP Call response is a great WordPress Plugin that lets you quickly set up a widget or a form for the visitors on your WordPress website. Visitors can then make use of the same and request a callback. The plugin can also send you an email every time a visitor shares details.

== Demo and Tutorial ==
[youtube https://www.youtube.com/watch?v=C7Z35dhhNNk]

Here a few features listed:
* Widget and Form to get contact requests.
* Shortcode to create a form on any page.
* customizable input fields and texts i.e you can show or hide input fields according to your requirement.
* Get an email notification for requests.
* Use avatars with a button created with a shortcode
* Functionality to manage all requests.

== Installation ==

1. Upload the `wp_call_response_bonus` folder to your `/wp-content/plugins/` directory
2. Activate the plugin using the `Plugins` menu in WordPress
3. `Call Response` menu from your sidebar.
4. For the first time, you need to provide your name and email to get an activation code.
5. The process is Complete
